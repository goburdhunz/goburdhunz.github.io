projects

Project 1: Build a vanilla JavaScript game with CSS grids - Solo project

We had a brief to make use of vanilla JavaScript and Dom manipulation to build a game. I chose to build Frogger - a classic arcade game from the 80s.

We had 5 days to build and we could not use HTML canvas but instead had to create a CSS grid to house the game. It was very interesting to use DOM manipulation to create a game and to mimic movement with players element and obstacle elements.

Project 2: 48-hour Hackathon - Front-end app using public API(s) - Pair Project

After using React for the second module, we were tasked with building a React front-end which consumed a public API. This was a pair project, with only one laptop between us. We chose to consume the public API from Listen Notes, a podcast library website.

We built a homepage which had a search function which made an API call with the search query and displayed the results within a card component. We also built a details page for individual podcasts, a random podcast selector which again utilised a specific endpoint from Listen Notes.

Project 3: MERN-Stack web app - React/Node/Express - Group project

We used React, Node.js/Express and MongoDB in the third module and we were briefed with building a MERN-stack web app with authentication, database models, CRUD routes and register/login. We worked in groups of 4 and had to set up a repo on Github and fork individually.

I chose to work on the Express back-end, setting up the back-end of the app; controllers, routing, mongoose models and basic token authentication. I also implemented a modal display on the front-end using a specific NPM package component and implemented comments on the details page of the app.

Project 4: Full stack web app with front-end in React/Back-end in Python/Django - Solo project

After a one week introduction to Python and Django, we were tasked with building a web app with a React front-end and Django back-end.

I built an app which was user focused and combined a blogging functionality with tailored news, job and events services. Users could register, login, create a blog, view all blogs, and when logged in - view personalised content based on their profile preferences.

We only had one week for this project but it was a great learning experience, especially for the Django backend, SQLite and database migrations.
